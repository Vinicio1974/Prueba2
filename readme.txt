# Instrucciones de ejecución

1. Requiere Java (JDK 11+) y Maven instalados.
2. Clonar el repositorio:
   git clone https://github.com/usuario/demoblaze-karate-api-test.git
3. Entrar al directorio:
   cd demoblaze-karate-api-test
4. Ejecutar las pruebas:
   mvn test
5. Los reportes se generarán en:
   target/karate-reports/index.html
